# 🚀 Programming Languages by Example

Welcome to our ultimate learning hub! Whether you are just starting your coding journey or looking to brush up on advanced algorithms, you've come to the right place. Our journey begins with a simple:
```c
printf("HELLO WORLD! 😎");
```
This repository is a hands-on, practical guide to learning programming languages through **real examples, algorithms, and exercises**. We currently feature comprehensive examples in **C**, **Python**, and **Lua**!

---

## 🛠️ C Programming
Dive deep into the core of computer science! Our C section takes you from the absolute basics to advanced memory management and fun projects.

**What's inside?**
* **The Basics:** Variables, Input/Output, Loops (`for`, `while`, `do-while`), and `if/switch` statements.
* **Intermediate Concepts:** Arrays, Strings, Boolean logic, Enums, and Variable Scope.
* **Advanced C:** Deep dive into **Pointers** (with tons of exercises!) and Memory Addresses.
* **Algorithms & Extras:** Binary Search, Bubble Sort, Fibonacci, Factorial, Towers of Hanoi, and even a fully playable **Snake Game**! 🐍

## 🐍 Python Programming
Explore Python, the powerhouse language known for its elegance and readability. This section heavily focuses on algorithms, data structures, and architecture.

**What's inside?**
* **Data Structures:** Linked Lists (Singly & Circular), Binary Trees (Insertion & Traversal), and Heaps (Max Heap).
* **Algorithms:** Recursive Functions, Binary Search, and Bubble Sort.
* **Object-Oriented Programming (OOP):** Classes, Inheritance, and Polymorphism.
* **Advanced Topics:** Multithreading and custom Set implementations.

## 🌙 Lua
A lightweight, fast, and powerful scripting language! Perfect for game development and embedded systems.

**What's inside?**
* **Getting Started:** Saying Hello to the World (and the Moon!).
* **Core Concepts:** Lexical Conventions, Types, and Logical Operators.
* **Data Handling:** Mastering Lua **Tables**.
* **Functions:** Writing clean and reusable code blocks.

---

### 💡 How to Use This Repository
1. **Browse by Language:** Pick the language you want to learn from the root directories.
2. **Follow the Numbers:** The folders are numbered chronologically (e.g., `1.Say Hello!`, `2.Variable`) to guide you step-by-step.
3. **Run and Tweak:** Don't just read the code—run the `.c`, `.py`, and `.lua` files, change the variables, and see what happens!
